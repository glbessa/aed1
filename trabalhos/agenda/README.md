## Proposta

Elaborar uma agenda sem o uso de variáveis, somente ponteiros poderão ser utilizados. Todas as variáveis necessário devem estar em um pBuffer ou em uma estrutura do tipo heap.

## Como executar

Compilar: `make all`

Executar: `make run`

Testar: `make test`
