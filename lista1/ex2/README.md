## Proposta

Faça um programa que armazena nomes em uma mesma string, que deve crescer ou diminuir conforme os nomes forem colocados ou retirados. Deve-se apresentar o seguinte menu ainda:
1) Adicionar nome
2) Remover nome
3) Listar
4) Sair

## Como executar

Compilar: `make all`

Executar: `make run`

Testar: `make test`
